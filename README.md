# 🧠 What are Molecular Descriptors?

Molecular descriptors are numerical values that represent the structural and chemical properties of molecules. They help convert chemical information into machine-readable features for predictive modeling.

Examples include:
- Molecular weight
- Topological indices
- Hydrogen bond descriptors
- Surface area descriptors
- Chemical fingerprints

These descriptors improve the capability of machine learning models in healthcare and pharmaceutical applications.
